Online Fashion Shopping

08 / 10  ·  Current Limitations

Current Limitations

📡

No Real-Time Data

Product prices are stored in a static JSON file and are not fetched live from retail store APIs in real time.

🗄️

No Backend / Database

The platform is fully frontend-only. There is no server, database, or authenticated data persistence.

👤

No Real User Accounts

Authentication is mocked. There is no real login system, profile management, or cloud-saved wishlist.

🏪

Limited Store Coverage

Currently compares 4 stores only (Myntra, Amazon, Flipkart, Ajio). More platforms can be integrated.
